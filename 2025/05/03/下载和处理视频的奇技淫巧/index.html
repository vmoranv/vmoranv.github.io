下载和处理视频的奇技淫巧

引言

本文汇集了我在下载和处理视频过程中遇到的一些奇技淫巧。

下载视频

  1. Youtube:

    1. https://www.socialplug.io/free-tools/youtube-video-downloader
    2. https://yt1d.com/en12/

  2. 猫抓等 m3u8 插件

  3. 网址前加 wn.run/ 进入相关下载站

  4. yt-dlp:

    1. yt-dlp/yt-dlp: A feature-rich command-line audio/video downloader (github.com) 下载对应系统工具

    2. FFmpeg 下载全量包并配置环境变量

    1. Chromium 系的浏览器需要导出cookies

导出浏览器 Cookie

方法一：使用浏览器插件 (推荐)

使用浏览器插件是导出 Cookie 最便捷的方式之一。以 Cookie-Editor 插件为例：

  1. 在您的浏览器（Chrome, Firefox, Edge, Opera 等）的应用商店或扩展商店搜索并安装 Cookie-Editor 插件。

    • 参考教程：手把手教你浏览器导入和导出cookie - CSDN博客

  2. 访问您需要导出 Cookie 的网站。

  3. 点击浏览器工具栏上的 Cookie-Editor 插件图标。

  4. 在插件界面中，您可以查看、编辑、删除当前网站的 Cookie。通常插件会提供导出功能，例如导出为 JSON 或其他格式。

  5. 选择导出选项，将 Cookie 保存到本地文件。

方法二：通过浏览器开发者工具

您也可以通过浏览器自带的开发者工具手动查找和复制 Cookie，但这通常比较繁琐，且可能无法导出所有需要的 Cookie。

  1. 打开您需要导出 Cookie 的网站。
  2. 按 F12 打开开发者工具。
  3. 切换到 Application (应用程序) 或 Storage (存储) 标签页。
  4. 在左侧菜单中找到 Cookies (Cookie) 项，展开后选择对应的网站域名。
  5. 您可以在右侧看到该网站的所有 Cookie 列表。手动复制所需的 Cookie 信息。

注意： 导出的 Cookie 文件可能包含敏感信息，请妥善保管。
