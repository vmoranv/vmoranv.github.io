Win各版本区别

  1. 家庭版 (Home)
  • 目标用户: 普通消费者和家庭用户。
  • 功能: 提供基本的Windows功能，如Cortana语音助手、Edge浏览器、Windows Hello（生物识别登录）等，但缺少一些高级功能。
  • 适用场景: 适合日常使用，如上网、观看视频和处理文档。
  1. 专业版 (Professional)
  • 目标用户: 技术爱好者和小型企业。
  • 功能: 包含家庭版的所有功能，并增加了如BitLocker全盘加密、远程桌面、组策略管理等企业级功能。
  • 适用场景: 适合需要更高安全性和管理能力的用户，如IT专业人员。
  1. 企业版 (Enterprise)
  • 目标用户: 大型企业和组织。
  • 功能: 在专业版基础上增加了更多安全和管理工具，如DirectAccess、AppLocker、Windows Defender高级威胁防护等。
  • 适用场景: 适合需要大规模管理和安全控制的环境。
  1. 教育版 (Education)
  • 目标用户: 学校和教育机构。
  • 功能: 基于企业版，提供与教育相关的特定管理功能，支持学校的教学需求。
  • 适用场景: 适合学校教职工和学生使用。
  1. 专业教育版 (Pro Education)
  • 目标用户: 教育机构中的技术人员。
  • 功能: 在专业版基础上增加了教育特定的管理控件和设置。
  • 适用场景: 适合需要在教育环境中使用高级管理功能的用户。
  1. 专业工作站版 (Pro for Workstations)
  • 目标用户: 高性能计算需求的专业用户。
  • 功能: 支持高端硬件配置，提供更快的数据处理能力，支持ReFS文件系统和高达4路CPU及6TB内存。
  • 适用场景: 适合需要进行图形设计、大数据分析等高负载工作的用户。
  1. LTSC版 (Long-Term Servicing Channel)
  • 目标用户: 企业用户需要长期稳定环境的场合。
  • 功能: 不包含某些常规更新（如新功能），只提供安全更新，适合对系统稳定性要求高的业务环境。
  • 适用场景: 适合医疗、金融等行业。

LTSC为长期支持版本,更多非长期办公所需的组件都被精简或压缩,在调用时会出现即时编译压缩组件或频繁调用不存在的组件,这会导致更多性能损失;相比之下,Pro更适合个户日常使用.

内存压缩

什么是 Windows 内存压缩，如何启用和禁用 (sysgeek.cn)

Windows中的Swapfile.sys是什么？ - 系统极客 (sysgeek.cn)

Tip:Windows 10 Version 1507引入特性;

原理:

​ 分页:将RAM中不活跃的数据压缩进C:\pagefile.sys,与Swapfile共同管理内存,这依赖硬盘读写性能;而内存压缩就把RAM中不活跃的数据压缩到内存而非硬盘中,后续使用更快的内存进行XPRESS解压处理,但这会占用RAM空间,同时所有压缩与解压过程都需要消耗CPU资源;

启用或禁用:

  1. 查看内存压缩策略:Get-MMAgent或任务管理器->性能->内存->(已压缩)
  2. 禁用 Windows 内存压缩:Disable-MMAgent -mc
  3. 启用 Windows 内存压缩:Enable-MMAgent -mc
  4. (Recommended)Windows11轻松设置->其他设置

建议:

  • 对于 RAM 有限的旧设备，内存压缩可以为更多应用程序和进程腾出 RAM 空间，但也需要考虑 CPU 性能和负载。
  • 新一代的 CPU 可能会高效处理压缩/解压过程。然而，如果内存管理过程消耗了大量 CPU 资源，也可以考虑禁用内存压缩。
  • 如果已经有足够的 RAM，但内存压缩导致设备出现性能问题，也可以选择禁用它。
  • 内存小于 4G 的 Windows 最好不要禁用虚拟内存
  • 一般情况下，可以把虚拟内存大小设置为实际物理内存的 1.5 倍
  • Windows 中增大虚拟内存会立即生效，减小和禁用虚拟内存需要重启系统
  • CPU性能较弱且内存充足推荐关闭：例如轻薄本用户追求续航和低发热
  • 专业数据处理需求推荐关闭：如科学计算、实时图像处理等需要CPU专注的场景
  • 内存容量较小（≤8GB）保持开启：压缩可有效缓解内存压力，避免频繁分页
  • 多任务或大型软件使用保持开启：如视频编辑、3D建模等场景，压缩能提升整体流畅度

4060Laptop OC

  1. MSI Afterburner->设置->常规->兼容性->全选;
  2. MSI Afterburner->Core Voltage/Power Limit/Temp Limit拉满;
  3. 开GPU-z检测,甜甜圈短时烤鸡,GPU Votage则为电压甜点值;
  4. Memory Clock:初始+1000Mhz,微调至极限过测,后续追求稳定降600-400Mhz,极限降200-100Mhz
  5. Core Clock:初始+300Mhz->Voltage/Frequency curve editor找到电压甜点值,左边提300Mhz,右边持平,极限过测即可,每次升/降60Mhz
